关于方程是否有解的另个描述

对于方程组 AX = B，对A进行消元，如果A出现了0行，0行对应的增广矩阵中的B的元素的组合也必须是0,方程组才会有解
对于方程组AX = b,对A消元得消元矩阵U，U中的自由列都取0，可得方程组的特解，再求AX = 0.即方程组的零空间，然后将特解 + 零空间上的任意向量可得方程组AX = b的通解

关于矩阵满秩

行列相等，都满秩：AX = b 有一个解
列满秩，行有剩余：AX = b有一个b的特解或者没有解
行满秩，列有剩余：AX = b总是有解
都不满秩：无解（b不在0空间上）或者无穷多个解
在几何上满秩就是这个矩阵的列空间 = 向量维数的基空间

线性相关性

对于方程组 Ax1 + Bx2 + Cx3 +……+QxN = 0有解，那这些未知量就是线性相关的，否则就是线性无关的
对于向量C1A + C2B + ….CnN，当他们的常数系数Cn不为0时，向量的何为0有解，那么这些向量就是线性相关的

空间与基向量

对于给定空间：

基向量的个数就是空间的维数（n个向量）

列空间

对于一个矩阵，秩的个数（r）就是这个矩阵列空间的维数（线性不相关的列数），所以列空间的基就是主列，维数就是秩r，列变换不会改变列空间，但会改变行空间

零空间

对于一个矩阵，自由列的数量就是零空间的维数（计算0解的时候，0空间的列数取决于自由列的数量），所以Ax = 0的解构成了0空间的基，0空间的维数是（n（列数） - r）,求零空间的基就是试着寻找一个产生零列向量的列组合

行空间（矩阵转置的列空间）

行向量生成的空间为行空间，处理上可以将矩阵转置然后处理列向量，得到的矩阵的行空间，行空间和列空间的维数是相同的，所以行空间的秩和维数也是r，行变换不会改变行空间，但会改变列空间，所以对只有行变换矩阵消元后得到的行就是行空间的基

左零空间（转置零空间）

矩阵的转置的零空间，左零空间的维数是（m（行数也就是转置后的列数） - r），相对于零空间，求左零空间就是试着从矩阵正找一个产生零行向量的行组合

矩阵空间

同向量一样，一个矩阵的集合，如果这个矩阵里面任意矩阵相加后还在该集合中，那这个集合就是一个矩阵空间
